Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ: __[77 78 79]__
```
Мы создали массив a из 5 элементов, затем сделали слайс b с 1 по 3 элементы массива a включительно.
```
